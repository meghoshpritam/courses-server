## Get Started

```bash
# clone repo
git clone https://github.com/meghoshpritam/courses-server.git
# go to the repo
cd courses-server
# install dependencies
npm install
# start the project
npm start
```

- open in your favorite browser by going [localhost:4000](http://localhost:4000/)
